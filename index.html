Pokémon Orange & Blue

Pokémon Orange & Blue is a version of the Pokémon games themed around Bucknell University. It is written in Java and was created by myself and 3 others in the spring of 2016 for a Software Engineering and Design class.

Our goal was to create something that looked and felt like one of the original Pokémon games. I was in charge of creating the graphical interface and its interactions with the underlying components of the game.

Physio

Physio is an android application that acts as means of communication between a Microsoft Band and a remote server used to gather a user's physiological signals. The gathered physiological data can then be fed to a cognitive architecture that uses machine learning techniques to create a cognitive model personalized to the user.

The application itself is written in Java and uses HTTP POST requests authenticated with Java Web Tokens to communicate with the companion server, written in Node.js with a MongoDB database.

There's an article about it!

Dime

Dime is an iOS application designed to simplify sending expense reports. Its main goal is to make expense reports easy by not requiring a third-party web account to compile or send a group of expenses as a report.

Dime compiles recorded expenses into PDFs, generating them on the device itself, meaning no internet is required to push a report to the user's outbox.
